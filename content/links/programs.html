Don't be without these...

These are the most essential utilities that every computer user should have. Where possible, to keep things simple, I've favoured cross-platform applications, so now you have no excuse.

Office suites and word processing

http://libreoffice.org/ http://openoffice.org/ http://abisource.com/

Stop throwing money down the swanney buying Office software. Libre Office is a fully functional office suite, compatible with all open and proprietary document and presentation formats. The same can be said for OpenOffice, which has come home after a spell with Oracle, and is now an apache project.

Abiword is a much lighter alternative for those not needing a full office suite, or using older, less powerful computers, and also fully supports all formats.

Jitsi

http://jitsi.org/

Shake off the shackles of Skype, and replace it with an encrypted fully functional and compatible alternative. Desktop sharing etc. is included.

Because development is ongoing, install a stable build, and not a nightly. Downloads are clearly identified as one or the other.

Bleachbit

http://bleachbit.com/

Imagine a kitchen where nobody cleans up after themselves. Not a pretty picture, but this is what almost every program on your computer is doing... image viewers, video thumbnailers etc. etc.

Now imagine what the picture will be like after a week, a month... how long since you bought that thing? In any case, time for a cleanup. Less files means faster file retrieval, before we even touch the issue of privacy.

Noscript

http://noscript.net/

I once wanted to find a phone number online, and had some naive expectation that the request would be quick and simple. It should have been, but it seemed like 10 sites all wanted to analyse every keystroke. This delayed the response considerably.

My tip for using this is to analyse what's being blocked, especially when visiting a frequently used site for the first time. Youtube thumbnails, for example, won't show until you allow ytimg which, when you think about it, means Youtube images. Expect similar on other sites, but the result is well worth it.

Request Policy

https://addons.mozilla.org/en-US/firefox/addon/requestpolicy/

Tame those websites that redirect who they want to where they want, to run javascript for God knows who, or for what goal. With this, each permission to redirect has to be explicitly granted, even for search engine results!

Typically I use this to block analytics and speed up page loading. It does take a little getting used to, but in my opinion, it's well worth the effort.

MaskMe

https://www.abine.com/maskme/

This is one I like a lot, and it's only a taster of what's available from the same place. This gets around the problem of nosey websites asking for an email address or phone number you don't want to give.

As a bonus, it will generate and remember secure passwords for you. There are options to view the secure passwords, sync them for use on multiple browsers or devices.

Ghostery

http://ghostery.com/

This is is an alternative to the noscript and requestpolicy that I use, and many users will probably find it simpler to use than using two tools.

There are versions available for all the major browsers, which isn't true about the other tools here which are primarily for Gecko browsers like firefox.
