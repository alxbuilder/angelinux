Get started with Docker Containers

Virtualisation is by any standards, cool technology. The ability to fire up disposable hardware is very useful for tests in an isolated environment but has significant overhead, but docker significantly reduces this overhead by sharing the kernel.

Many moons ago, I researched linux containers and UnionFS filesystems. UnionFS is well described on Wikipedia". Both show up as key components in Docker.

Why is Docker cooler?

Traditional virtualisation overheads are high. All hardware has to be recreated. Because Docker is a container technology, it simply re-purposes the already running infrastructure, mapping itself into the container, eliminating duplication. As a result, Docker containers are much more lightweight and faster to deploy on any platform.
