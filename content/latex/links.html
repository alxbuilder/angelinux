Cross-references

  • Anything with a number can be referenced by \label{name}
  • If there are many, it's best to indicate the type:
    • \label{sec:survey}: Section
    • \label{th:pifagor}: Theorem
    • \label{eq:gdp}: Equation
    • \label{fig:tree}: Figure
  • Inclusion of showkeys package makes these visible in the final PDF.
    • These can later be referred to with one argument like:
    • \ref{name}: number of the object
    • \pageref{name}: number of the page
    • \eqref{name}: number of the equation

Packages and what they provide

  • varioref: Improves cross-referencing by allowing:
    • Referencing links on the current page without page number
    • Referencing links on the previous and next pages without page number
  • hyperref: Allows creation of clickable, live hyperlinks in the final PDF.
