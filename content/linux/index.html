What to do now...

  1. Wipe the disk

    If you have ever done shopping on your PC, then your digital rubbish is going to be still on your hard disk, so DO NOT DISCARD IT before wiping the disk clean, for the very same reason as you shred confidential documents.

  2. Here's how

    Use Darik's Boot and Nuke to create a bootable live CD which you can boot your computer from. This will present a menu of options, all of which work, that allow a total forensic wipe of your disk, bringing it back to factory condition. This same utility is available on the System Rescue CD, which no self-respecting computer user should be without!!

  3. The bad fix, speeding up XP

    First off, this is a VERY BAD IDEA as XP is no longer supported so there will be no future security fixes, or defence against malicious software, viruses, trojans and worms. So for those who absolutely insist, you have been warned, and consider this fix a temporary measure.

    The way I explain this is to get people to imagine a kitchen where nobody cleans up. After one meal, there's a mess. Now imagine it after a day, a week... a year? Every web page, gallery, flash-commercial etc. leaves it's crumbs behind on your disk. This may have gone on from the very time the computer was bought.

  4. Wipe private data from you disk...

    1. Clean up temporary files

      Get Bleachbit (my preference, and it's cross-platform). This can drastically reduce the number of files on the disk. Less files means that the files you want to open can be much more easily found.

    2. Defragment the hard disk

      To defragment a hard disk under windows, use Windows explorer to navigate to the disk (usually C: drive). Once there, right click and find defragment on the memu.

      Note: Be prepared to run this overnight, and don't expect the computer to be usable while this is running.

    3. Install Linux

      There are numerous linux distributions suitable for use on computers of all ages and capabilities. Many come as Live CD s, many of which are installable.

      The handiest site for getting Linux or Linux Live CDs is Distrowatch.
